# 需求清单

项目所使用的编码格式：UTF-8

## 产品定位

任务Todolist + AI Prompt管理器，用于人类使用者拆分任务以及管理对应的AI Prompt。
核心差异化：**任务驱动的Prompt管理**，将任务与Prompt关联，比单纯的Prompt收藏夹更有场景价值。

技术栈：纯前端实现

---

## 功能清单

### 一、基础架构（必须实现）

- [ ] **数据持久化**：使用 localStorage 或 IndexedDB 保存数据，确保刷新页面数据不丢失 [必须]

- [ ] **导入/导出**：支持JSON格式导出备份，方便数据迁移和恢复 [高优先级]

---

### 二、界面与交互

- [ ] 要求界面简约，动画风格和界面设计使用苹果风格完成 [高优先级]

- [ ] **深色/浅色模式切换**：支持明暗主题，符合苹果设计规范 [中优先级]

- [ ] **快捷键支持**：`Ctrl+N` 新建任务、`Ctrl+Enter` 完成任务等常用快捷键 [低优先级]

---

### 三、任务管理（Todolist）

- [ ] 拥有计划todolist功能，可以管理各个任务，当完成后用户手动点击勾选，反馈动画需要给予用户较强的反馈和爽感 [高优先级]

- [ ] **任务分组/项目**：支持按项目归类任务，如"论文写作"、"代码开发"等 [中优先级]

- [ ] **子任务支持**：大任务可拆分为多个子任务，支持层级管理 [中优先级]

- [ ] **拖拽排序**：支持拖拽调整任务顺序和优先级 [中优先级]

- [ ] **任务状态扩展**：除完成/未完成外，增加"进行中"状态 [低优先级]

- [ ] **搜索/过滤**：支持按关键词搜索任务，按状态过滤 [低优先级]

---

### 四、Prompt管理

- [ ] 区别于普通todolist，其中需要给用户留出prompt输入框，新建list的时候不需要同时加入prompt，允许用户先创建list后添加prompt [高优先级]

- [ ] **一键复制Prompt**：点击按钮即可复制Prompt到剪贴板，带成功提示 [高优先级]

- [ ] **Prompt模板库**：预设常用Prompt模板（如代码审查、文案润色、翻译等），用户可快速套用 [中优先级]

- [ ] **变量占位符**：Prompt中支持 `{{变量}}` 语法，复用时快速替换关键内容 [中优先级]

- [ ] **Prompt版本历史**：记录Prompt修改历史，方便回溯查看 [低优先级]

---

### 五、AI功能（后期扩展）

- [ ] 预留ai润色prompt功能，后期实现用户自定义接入api实现prompt智能润色 [低优先级]

---

## 优先级说明

| 等级   | 含义              |
| ---- | --------------- |
| 必须   | 产品基础功能，不实现则无法使用 |
| 高优先级 | 核心功能，第一版必须包含    |
| 中优先级 | 重要功能，建议第一版包含    |
| 低优先级 | 增强功能，可后续迭代添加    |
